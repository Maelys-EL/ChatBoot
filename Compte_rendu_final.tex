\section{Introduction}

Pour réaliser ce rendu, nous avons choisi de réaliser un projet fictif, mais qui reste relié à l'une de nos matières de ce semestre : "Introduction à l'IA". Nous avons donc pris le 2e TD de que nous avons eu dans cette matière pour l'utiliser en tant que base de ce faux projet. Le but du TD était de créer un petit chatbot, capable d'agir en fonction de règles faites avec des expressions régulières.
Notre faux projet est donc le suivant : Créer un petit chatbot et faire une brève introduction sur l'histoire de l'IA pour pouvoir faire une séance d'initiation à l'IA dans un collège. Le projet sera accompagné d'un site web.
