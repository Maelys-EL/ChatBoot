\section{Introduction}

Pour réaliser ce rendu, nous avons choisi de réaliser un projet fictif, mais qui reste relié à l'une de nos matières de ce semestre : "Introduction à l'IA". Nous avons donc pris le 2e TD que nous avons eu dans cette matière pour l'utiliser en tant que base de ce faux projet. Le but du TD était de créer un petit chatbot, capable d'agir en fonction de règles faites avec des expressions régulières.

\section{Réalisation du projet}

[B1] Pour le projet, nous avons décidé de faire une branche pour chaque partie du projet. La branche main pour le code Python et le document sur l'histoire de l'IA, la branche "Compte\_rendu" pour ce document et la branche "Site" pour le site web.
Nous avons commencé par créer la branche "Compte\_rendu" en ligne de commande.
On commence par récupérer le dépôt distant avec la commande :
\begin{verbatim}
    git clone [URL du dépôt]
\end{verbatim} 
Puis on se déplace sur le premier commit pour créer la branche à partir de là, on utilise donc les commandes : 
\begin{verbatim}
    git checkout [SHA-1 du 1er commit] 
    git branch Compte_rendu
\end{verbatim} 
Enfin, on se déplace sur la branche avec la commande :
\begin{verbatim}
    git checkout Compte_rendu
\end{verbatim}

[B2] Comme dit précédemment, nous avons choisi de faire une seule version du projet : la version finale. Il n'y a pas eu de version intermédiaire donc nous avons seulement créé un tag pour les commits finaux. 
Pour créer un tag, nous avons fait comme suit :
\begin{verbatim}
    git log (pour récuperer le nom du commit sur le quel mettre le tag)
    git tag -a vFinal [numéro du commit]
    
    (Ouvre un éditeur de texte dans le quel écrire le message du tag, ':w' 
    pour sauvegarder et entrée pour quitter)
    git show vFinal (pour vérifier sur quoi porte le tag)
    git push origin vFinal
\end{verbatim}

[C3] Pour finir, il nous fallait mettre en place les submodules. Nous avons seulement utilisé la bibliothèque pandas dans notre code donc nous avons rajouter un seul submodule au projet avec ces commandes : 
\begin{verbatim}
    git submodule add https://github.com/pandas-dev/pandas
    git add .gitmodules pandas/
    git commit -m "Ajoute du submodule"
    git push origin main
\end{verbatim}
Ici on ajoute le submodule puis on l'ajoute dans l'index avec le dossier cache .gitmodules. Puis on fait un commit et on met à jour le dépôt distant en faisant bien attention à ce que le dépôt local soit mis à jour avec le dépôt distant.
